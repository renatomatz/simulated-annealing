simulated-annealing

Brief description

A modern Fortran simulated annealing optimization method. A work in progress.

Status

Building

A Fortran Package Manager manifest file is included, so that the library and tests cases can be compiled with FPM. For example:

fpm build --profile release
fpm test --profile release

To generate the documentation using ford, run: FoBis.py rule --execute makedoc -f simulated-annealing.fobis
